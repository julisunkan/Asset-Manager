# ROLE

You are a senior full-stack software engineer, UI/UX designer, audio DSP engineer, and AI engineer.

Build a production-quality web application called **DJ & Mixer**.

The application should have a modern professional interface inspired by Serato DJ, Rekordbox, VirtualDJ, and djay Pro while remaining simple enough for beginners.

Write clean, modular, scalable, production-ready code with comments and documentation.

---

# MOST IMPORTANT REQUIREMENT

This application is **100% privacy-first**.

## ABSOLUTE RULES

- Music/audio files MUST NEVER be uploaded to the server.
- Music files MUST NEVER leave the user's device.
- Flask MUST NEVER receive audio files.
- Do NOT create upload endpoints.
- Do NOT save temporary audio files.
- Do NOT perform server-side audio processing.
- Do NOT use cloud AI for audio analysis.
- Do NOT send music through WebSockets.
- Do NOT transmit audio in any way.

The Flask backend should ONLY serve:

- HTML
- CSS
- JavaScript
- Images
- Icons
- Fonts
- Static AI model files (optional)

All music analysis, playback, mixing, effects, AI inference, rendering, and exporting must happen entirely inside the user's browser.
